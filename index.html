Time of Day

A JavaScript project by Mohamed Deria

Greeting Goes Here

This page demonstrates how we can use JavaScript to aquire client information (which in this case is time) in order to project on the website a certain color, picture, and greeting depending on the time of day. This is a basic page but the idea behind the javascript code implemented here could result in creating web pages that can change between dark mode and light mode to change brightness on the screen depending on the certain time of day.
